The view class for handling all of the api-con-mngr's html data.

Used when the api manager has to show pages in wp-admin/admin-ajax.php.

Usage:

//construct and set $obj = new API_Con_Mngr_View(); $obj->body[] = "

My lovely header

"; $obj->body[] = "

no more hello world

"; $obj->body[] = "

hellow universe ;)

";

//to print html to client $obj->get_html();

//to return html as string $html = $obj->get_html( false );
